Exporting the report

 You can export your report or any search results to MS Word, MS Excel, or to HTML using the export button.

Tip:    If you intend to copy an exported table to Microsoft Outlook so that you can email it, ensure that HTML is chosen in the "Send in this message format" field on the Mail Format tab of the Options dialog box in Outlook.

Exporting to Crystal Reports (RPT)
  • Once you are on the Export Options page, choose Crystal Reports (RPT) in the drop-down list and click the OK button.

    You will be able to save your new report or open it.

Exporting to Adobe Acrobat (PDF)
  • Once you are on the Export Options page, choose Acrobat Format (PDF) in the drop-down list and click the OK button.

    You will be able to save the PDF or open it.

Exporting to Microsoft Excel (XLS)
  • Once you are on the Export Options page, choose MS Excel (XLS) in the drop-down list and click the OK button.

    You will be able to save the new spreadsheet or open it.

Exporting to Microsoft Excel, Data only (XLS)
  • Once you are on the Export Options page, choose MS Excel - Data only (XLS) in the drop-down list and click the OK button.

    You will be able to save the new spreadsheet or open it.

    Note:    the data-only export does not preserve the exact layout and formatting of objects in your report.

Exporting to Microsoft Word (RTF)
  • Once you are on the Export Options page, choose Microsoft Word (RTF) in the drop-down list and click the OK button.

    You will be able to save the new document or open it.

Exporting to Microsoft Word - Editable (RTF)
  • Once you are on the Export Options page, choose Microsoft Word - Editable (RTF) in the drop-down list and click the OK button.

    You will be able to save the new document or open it.

    Note:    The difference between the two Microsoft Word formats is that Microsoft Word (RTF) exports your report to a Word document that uses an object frame for each report object, while Microsoft Word - Editable (RTF) does not use object frames—which makes it easier to edit the output.

Exporting to Rich Text Format (RTF)
  • Once you are on the Export Options page, choose Rich Text Format (RTF) in the drop-down list and click the OK button.

    You will be able to save the new document or open it.
