Printing the report

 To print a copy of your report, click the "Print this report" button.

The Crystal Reports Viewer - Web Page dialog box appears along with the Print dialog box. Use the Print dialog box to select a printer, page range, number of copies, and other options, and then click Print. The Web Page dialog box remains on the screen to track the progress of your print job.
