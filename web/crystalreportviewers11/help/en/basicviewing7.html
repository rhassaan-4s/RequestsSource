Step 3: Results

  • Click the Results tab.

Now the report shows the search table that lists the results of the Boolean search you just performed. If there is no information matching the criteria you entered, no search table is displayed.

If you want to save your results, you can export them by clicking the Save Results As button. For more information, see Exporting the report.

Note:    If you export a search table, you must close the export window before exporting the table again in the same session.
