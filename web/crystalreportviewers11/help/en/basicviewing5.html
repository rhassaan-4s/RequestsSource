Step 1: Select the fields from the report to display

  1. Click the Fields tab and select the fields you want displayed in your search results.

    2. Use the arrow buttons to move fields from the Available Fields list to the Fields to Display list. Once you have added fields to the Fields to Display list, you can change their order by using the up and down arrow buttons.

    Tip:    You can select multiple fields to move by using Shift-click or Ctrl-click.

  2. Go to Step 2: Select a condition for your results.
