The Group Tree

 The Group Tree provides a list of the values that the report data is grouped on. Clicking the Show/Hide Group Tree button shows or hides the Group Tree. To navigate to a particular group in your report, click the group in the group list in the Group Tree.

For more information, see the Crystal Reports Online Help (Crw.chm) and search for the topic "Grouping."
