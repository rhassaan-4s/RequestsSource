Searching for text

 You can search for any text string in your report. Type the string in the text input field and click the "Search for text" button beside the text box. The text "End of Search" appears in the text input field when the search has reached the end of the report.

Note:    

  • You cannot use wildcard characters, such as * or ?, in the text input field.
  • If you are on page two, and you search for something that is on page one of your report, the viewer will not find it. It can only search from the next page forward. So if you're on page three, it will only search from page four to the end of your report.
